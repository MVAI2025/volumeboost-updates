VolumeBoost Beta

Version 1.0.2

Interaction safety update for the direct-download beta.

What’s New

  • The popover action layout is safer and clearer for normal use.
  • Start/Stop Boosting is now the only large primary action.
  • Updates, Diagnostics, and Quit App now live in the smaller centered footer row.

Why It Changed

This reduces accidental app quits when users only meant to stop boosting.

Install

  1. Download the beta zip.
  2. Move VolumeBoost.app into /Applications.
  3. Launch the app and allow system audio access if macOS prompts.

Feedback

If you hit a problem, use Diagnostics from the VolumeBoost popover and include the zip with your report.
