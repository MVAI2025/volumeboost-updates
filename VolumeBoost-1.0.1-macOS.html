VolumeBoost Beta

Version 1.0.1

Polish update for the direct-download beta with live updates, tighter popover UX, and more reliable startup behavior.

What’s New

  • The updater is now live on the public direct-download track.
  • The app icon and menu bar icon were refreshed to match the approved branding.
  • The menu bar popover now has clearer primary actions and a smaller centered utility footer.

Improvements

  • Live boosting startup is more reliable after repeated stop and start cycles.
  • Inline help, update visibility, and footer alignment issues found during beta testing were fixed.
  • Direct-download release artifacts keep a cleaner public file name.

Install

  1. Download the beta zip.
  2. Move VolumeBoost.app into /Applications.
  3. Launch the app and allow system audio access if macOS prompts.

Beta Focus

  • Repeated start and stop behavior
  • Output switching and reconnects
  • Sleep and wake recovery
  • Low-volume content from browsers, streaming apps, and local media players

Feedback

If you hit a problem, use Diagnostics from the VolumeBoost popover and include the zip with your report.
