VolumeBoost Beta

Version 1.0.0

First notarized direct-download beta for quiet-audio boosting on macOS 14.2 and later.

What’s New

  • Live Core Audio taps path is now the default shipping backend.
  • Menu bar controls include boost, smart boost, limiter, launch at login, and diagnostics export.
  • Packaging, signing, notarization, and release zips now come from the Xcode app target.

Install

  1. Download the beta zip.
  2. Move VolumeBoost.app into /Applications.
  3. Launch the app and allow system audio access if macOS prompts.

Beta Focus

  • Repeated start and stop behavior
  • Output switching and reconnects
  • Sleep and wake recovery
  • Low-volume content from browsers, streaming apps, and local media players

Feedback

If you hit a problem, use Export Diagnostics from the VolumeBoost popover and include the zip with your report.
